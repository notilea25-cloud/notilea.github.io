## 🚀 Cómo publicar en GitHub Pages

1. **Subir la carpeta** a tu repositorio `notilea25-cloud/notilea.github.io`
2. Ir a **Settings → Pages**
3. En *Source*, seleccionar la rama `main` y la carpeta `/notilea-noticias` (o raíz `/`)
4. La página estará disponible en:
   ```
   https://notilea25-cloud.github.io/notilea-noticias/
   ```
   *(o en la raíz si copiás el `index.html` directamente)*
